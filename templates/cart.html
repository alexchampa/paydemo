Step: Shopping Cart

This page demonstrates the dynamic features of the cart and serves as the entry point into the various B2B checkout flows.

Real-Time Editing ✏️

You can edit the quantity for any item in your cart. All prices will update automatically without a page reload, and the changes are saved to your session in the background.

Dynamic Checkout Workflows ⚙️

When you click Checkout, you will be redirected to the login page if you aren't already logged in. After authenticating, the system triggers a different workflow based on your user profile:

  • jdoe: As a standard registered user, the purchase is approved without further verification.
  • fraud: This user is flagged and must go through an identity verification step before the purchase is approved.
  • otp: This user is prompted to enter a 4-digit code to simulate two-factor authentication.
  • reject: This user's transaction is automatically declined by the system.
  • guestcheckout: This user is guided through the full onboarding process, including OTP, company verification, and a final identity check.
